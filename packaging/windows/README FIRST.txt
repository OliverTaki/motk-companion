MOTK COMPANION — FIRST START
============================

New installation
----------------
1. Double-click "INSTALL MOTK COMPANION.cmd".
2. Choose the local media folder and camera control method.
3. Click "Save & Start".
4. Open Start > MOTK > "MOTK Companion - Copy Pairing Key".
5. In MOTK Shoot, open Settings > Camera > Tether, paste the key, and press Connect.

You do not have to close a running Companion before an install or update. The
installer stops only MOTK Companion, replaces it safely, and Setup starts it again.

Already installed
-----------------
Use the Windows Start menu:

- MOTK Companion — start the local service. Keep its window open while shooting.
- MOTK Companion - Setup — change the local media folder or camera method.
- MOTK Companion - Copy Pairing Key — copy the private local connection key.
- MOTK Companion - Open Local Media — open the exact folder used for local media.

What it does
------------
MOTK Companion gives the browser app permission to work with a selected local
folder and supported tethered cameras. It also runs large MOTK Media Tools jobs
locally. Media stays on this computer unless you explicitly configure an upload.

MOTK Shoot:
https://motk-public-site.pages.dev/apps/shoot/index.html

Keep the pairing key private. The public beta is unsigned, so Windows may warn
about the publisher. The CMD installer starts the bundled PowerShell scripts with
the required one-process permission without changing the PC-wide execution policy.
